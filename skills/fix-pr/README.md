# fix-pr

## Autor
awnt9.

## Resumen
Diagnostica Pull Requests fallidas y propone arreglos conservadores antes de tocar código.

## Cuándo usarla
Cuando una PR falle checks, CI, lint, typecheck o tests.

## Cuándo no usarla
Cuando quieras cambios automáticos sin aprobación previa del usuario.

## Requisitos previos
- Acceso al repositorio, GitHub CLI disponible y permisos para leer la PR.
- acceso a github cli y permisos para leer la PR.

## Cómo usar
Escribe <code>&#47;fix-pr</code> en el chat.
