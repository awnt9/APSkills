# react-flat-component

## Autor
Pendiente de definir.

## Resumen
Crea componentes visuales React de estilo flat en formato CodePen.

## Cuándo usarla
Cuando necesites generar un componente React visual, editable y autocontenido con HTML, CSS y JSX.

## Cuándo no usarla
Cuando necesites una aplicacion completa, assets externos, Tailwind, canvas o animaciones automaticas no solicitadas.

## Requisitos previos
Un entorno tipo CodePen con React, ReactDOM y Babel habilitados para JSX.

## Cómo usar
Describe el objeto visual, la interaccion opcional y las restricciones; la skill devuelve los paneles HTML, CSS y JSX.
