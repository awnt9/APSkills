# prepare-skill

## Autor
awnt9.

## Resumen
Normaliza borradores de skills para que encajen con la estructura y validaciones del repo.

## Cuándo usarla
Cuando un contributor haya creado una skill incompleta, fuera de carpeta o en formato incorrecto.

## Cuándo no usarla
Cuando quieras reescribir el contenido funcional de la skill en vez de conservarlo.

## Requisitos previos
Un repositorio AP-Skills con cambios locales que incluyan una skill nueva o incompleta.

## Cómo usar
`prepare-skill`
