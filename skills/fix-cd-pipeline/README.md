# fix-cd-pipeline

## Autor
awnt9.

## Resumen
Diagnostica fallos de Continuous Deployment y prepara el arreglo local más pequeño posible.

## Cuándo usarla
Cuando falle un workflow, release job, despliegue o pipeline de CD.

## Cuándo no usarla
Cuando busques una auditoría preventiva completa en vez de arreglar una incidencia.

## Requisitos previos
- Acceso al repositorio, GitHub CLI autenticado y logs o runs de CD disponibles.
- acceson a github cli y permisos para leer el workflow, run o job de CD.
