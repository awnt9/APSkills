# best-practices-deployment

## Autor
awnt9.

## Resumen
Revisa despliegues con Docker, Docker Compose y GitHub Actions antes de producción.

## Cuándo usarla
Cuando quieras auditar riesgos y buenas prácticas de un sistema de despliegue de forma preventiva.

## Cuándo no usarla
Cuando solo necesites corregir un fallo puntual de logs sin revisión preventiva.

## Requisitos previos
- Un repositorio con configuración de despliegue, workflows, Dockerfile o Compose.
- Acceso a GitHub CLI y permisos para leer el repositorio y sus workflows.

## Cómo usar
``` /best-practices-deployment ```
