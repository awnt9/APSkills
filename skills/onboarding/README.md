# onboarding

## Autor
awnt9.

## Resumen
Genera un informe práctico y basado en evidencias para ayudar a un desarrollador a comprender un repositorio desconocido y empezar a contribuir con seguridad.

## Cuándo usarla
Úsala al incorporarte a un proyecto existente o al explorar por primera vez una base de código.

## Cuándo no usarla
No la uses como sustituto de una auditoría especializada ni cuando solo necesites resolver una duda puntual sobre un archivo conocido.

## Requisitos previos
Acceso de lectura al repositorio que se desea analizar.

## Cómo usar
Escribe <code>&#47;onboarding</code> en el chat.
