# pydantic-ai

## Autor

pydantic.

## Resumen

Skill para construir agentes con Pydantic AI, incluyendo herramientas, capabilities, salida estructurada, streaming, pruebas y patrones multiagente.

## Cuándo usarla

Úsala cuando el usuario mencione Pydantic AI, importe `pydantic_ai`, o necesite crear, probar o ampliar agentes basados en Pydantic AI.

## Cuándo no usarla

No la uses para la librería Pydantic de validación sin agentes, otros frameworks de agentes, o desarrollo Python general no relacionado con agentes de IA.

## Requisitos previos

Python 3.10+ y un proyecto que use o vaya a usar Pydantic AI.

## Cómo usar

Lee primero `v1/SKILL.md` y carga las referencias de `v1/references/` solo cuando la tarea las requiera.
