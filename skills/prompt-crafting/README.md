# prompt-crafting

## Autor
Pendiente de definir.

## Resumen
Recoge metricas de uso de prompts con skills en agentes de programacion.

## Cuándo usarla
Cuando quieras informar sobre consumo, llamadas internas y rutas usadas por una skill o prompt.

## Cuándo no usarla
Cuando no necesites medir ni resumir el gasto asociado a una interaccion con LLMs.

## Requisitos previos
Un agente de programacion ejecutando una skill o prompt que deba auditarse.

## Cómo usar
Escribe <code>&#47;prompt-crafting</code> en el chat después de una skill o prompt para revisarlo.
